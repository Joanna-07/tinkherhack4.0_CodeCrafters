Project Description
TruthLens is a cutting-edge, frontend-powered platform designed to help users analyze the credibility of news, social media posts, tweets, and WhatsApp forwards before sharing. 
With a visually immersive AI-style dashboard, TruthLens detects bias, emotional manipulation, clickbait, fact-risk, and logical inconsistencies, presenting results as interactive 
gauges, risk meters, and suggested actions.

The Problem statement
In today’s digital world, misinformation spreads faster than verified truth through:
News articles
Social media posts
Tweets
WhatsApp forwards
Most users lack quick tools to evaluate credibility before sharing content, leading to misinformation amplification.
TruthLens solves this by providing an interactive AI-style analysis dashboard that visually explains risk, bias, and manipulation signals in content.

The Solution
TruthLens is a frontend-powered AI simulation platform that analyzes user-submitted text and generates a professional credibility report with:
Bias detection
Emotional manipulation analysis
Clickbait probability
Logical fallacy detection
Propaganda technique identification
Fact-risk scoring
All results are displayed in a premium dark AI dashboard designed for clarity and engagement.
